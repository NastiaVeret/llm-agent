# AI Examiner Agent
Модель: Llama 3.3 70B

# Збереження даних
Формат: JSON

# Для цього проєкту використано локальні JSON файли замість бази даних.
Файли:
students.json
ім’я користувача
базова інформація

results.json
повна історія чату
оцінки за темами
фінальний результат

# Workflow
Ініціалізація
Користувач вводить ім’я та API‑ключ
Система обирає 3 випадкові теми start exam
Діалог
Користувач надсилає відповідь
Повна історія передається в ExaminerAgent
Рішення LLM
АБО задати наступне питання
АБО викликати transition_topic
Обробка Tool Call
Оцінка зберігається
Відбувається перехід до нової теми
Завершення

Після всіх тем викликається finish_exam

Результати зберігаються у results.json
